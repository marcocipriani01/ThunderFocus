ThunderFocus è un software open-source per il controllo di focheggiatori compatibili, sviluppato da Marco Cipriani. Il codice sorgente del progetto è reperibile su GitHub ed è sottoposto alla licenza software GNU General Public License versione 3. Utilizzando questo software si accettano tutte le condizioni della sua licenza.

Altri progetti alla pagina marcocipriani01.github.io!
